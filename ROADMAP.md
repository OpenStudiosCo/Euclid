# Euclid Roadmap

This roadmap outlines the current direction of the Euclid plugin.
It is not a strict timeline or guarantee of features.

## Current Focus (1.x)

The goal of the 1.x series is to keep Euclid simple, reliable, and focused on its core use case:

- Converting logos, icons, and simple graphics into SVGs
- Providing fast, predictable results inside the WordPress media editor
- Maintaining a minimal and intuitive UI

## Planned Improvements

These are areas currently being explored:

- Improved colour tracing quality (smoother curves, reduced noise)
- Customisation options for tracing settings
- Adoption of multi colour space tracing based on Inkscape
- SVG optimisation and file size reduction

## Potential Features

These ideas may be explored in future versions:

- SVG editor for composition adjustments
- Batch conversion
- WooCommerce integration (Premium): allow customers to upload images and customize vectorization for print-on-demand products.
- Support for additional tracing APIs and techniques

## Out of Scope (for now)

To keep the plugin focused and maintainable, the following are not currently planned:

- Full design or illustration tools
- Complex photo vectorisation workflows
- Replacing dedicated vector software

## Philosophy

Euclid aims to do one thing well:
Converts static logos, icons and simple graphics to SVG.

Features will be added carefully to avoid unnecessary complexity.
