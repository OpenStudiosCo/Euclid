Convert raster images into SVGs directly inside the WordPress media editor.

== Description ==

Euclid lets you convert raster images into scalable vector graphics (SVG) directly within the WordPress image editor.

Choose between two vectorisation modes:

* **Black & White** – clean, high-contrast vector paths (ideal for line art and silhouettes)
* **Colour** – multi-colour tracing designed for logos and simple graphics

This plugin is best suited for logos, icons, and flat illustrations.

== Features ==

* Convert images to SVG inside the media editor
* Black & white vectorisation using Potrace
* Colour vectorisation for logos
* Saves SVGs directly to the WordPress Media Library

== Usage ==

1. Go to Media → Library
2. Select an image
3. Click "Edit Image"
4. Click "Convert to Vector"
5. Choose a vectorisation mode
6. Click "Save SVG"

== Notes ==

* Best results are achieved with high-contrast images or flat logos
* Complex photos may produce unpredictable results

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/euclid`
2. Activate the plugin through the WordPress admin
3. Open an image in the Media Library editor to begin
